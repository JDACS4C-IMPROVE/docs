RandomForest-DRP
=================
Random Forest implementation for Drug Response Prediction

Model Architecture
--------------------
Random Forest is a bagging ML model that combines the predictions of multiple decision trees.

Feature Representation
------------------------

   * Cancer features:

      * Gene Expression


   * Drug features:

       * Mordred descriptors
