Developer Guide
====================

The improvelib and IMPROVE software stack has been designed with maximum flexibilty to allow for various applications and workflows.
Both IMPROVE internal members and external community members are welcome to contribute new applications and workflows.
IMPROVE internal members should create new branch with your name included in the branch name for development.
External community members can fork the `IMPROVE repo <https://github.com/JDACS4C-IMPROVE/IMPROVE>`_ for development and then create a pull request.
